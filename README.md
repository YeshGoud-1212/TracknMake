# 🔥 TrackNMake

**Track it. Make it real.**

A minimal yet powerful discipline tracker that visualizes your consistency using a heatmap and keeps your daily execution in check.

---

## 🚀 Overview

TrackNMake is a productivity web app designed to help you:

* Track your daily tasks
* Visualize your discipline using a heatmap
* Stay consistent with a fixed timetable
* Eliminate “zero days”

It combines task tracking + visual feedback to make your progress impossible to ignore.

---

## ⚡ Features

### 🏠 Home Page

* Displays a cold motivational quote every time you reload
* Dark, minimal, distraction-free UI

---

### 📊 Year Heatmap

* Full-year view (GitHub-style)

**Color Logic:**

* ⚪ Future days → White

* 🔴 Past unmarked days → Red

* 🟠 1–50% → Orange

* 🟡 51–80% → Yellow

* 🟢 81–100% → Green

* 🔥 Today is highlighted dynamically

---

### 📅 Timetable Tracker

* Fixed daily task system
* Mark tasks as Done / Not Done
* Tracks completion % for each day

---

### 🔄 Smart Task System

* Tasks act as a global template
* Any add/delete/rename updates:
  → Applies to all future days
  → Keeps past data unchanged

---

### 📈 Insights

* Daily completion percentage
* Streak tracking (>80%)
* Monthly performance overview

---

### 🎯 Wallpaper Mode

* Fullscreen minimal UI
* Heatmap-only view
* Perfect for keeping discipline visible

---

## 🧠 Philosophy

> “What gets tracked gets done.”

TrackNMake is built on one idea:
**Visibility creates accountability**

If you can see your failures, you can’t ignore them.
